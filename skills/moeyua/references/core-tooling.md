# Tooling

Preferred tools and configurations for development.

## Package Manager

- Use **pnpm** as the default package manager

## Build Tools

- Use **Vite** for frontend projects
- Use **tsdown** for library bundling

## Linting

- Use **ESLint** with `@antfu/eslint-config`
- No Prettier — ESLint handles formatting

## Key Points

- Always use the latest stable versions of tools
- Prefer zero-config or minimal-config tools
- Use TypeScript everywhere possible
